# TS-OneKeyProcess
简化对TS片源的初步处理操作，实现无人值守一键完成对批量TS片源的初步处理。

本工具用于一键自动完成对TS片源的初步处理：

① 调用eac3to分析轨道信息

② 调用eac3to提取音频轨道、图形字幕

③ 调用dgindexnv索引视频轨道

④ 读取avs模板生成avs文件

⑤ 调用caption2ass_pcr抽取ass字幕

⑥ 自动压制小档
